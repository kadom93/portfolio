About me

I started Frontend Development as a hobby. Every now and then I would learn something new and try it. After some time I realized that I was spending quite a lot of time on learning HTML, CSS and JavaScript through websites suchas FreeCodeCamp, Lynda, Udemy etc. This was when I decided that I could actually use my knowledge to earn money and start a career in web development.
I am a rather creative person so designing websites and then building them has not been a problem for me. Although I sometimes struggle to get a particular outcome, I never give up and try to find a solution using available resources.
